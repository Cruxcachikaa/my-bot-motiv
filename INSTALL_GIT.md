# Как установить Git на Windows

## Способ 1: Скачать установщик (Рекомендуется)

1. Зайди на [git-scm.com/download/win](https://git-scm.com/download/win)
2. Скачай установщик (автоматически определит 64-bit версию)
3. Запусти установщик
4. Нажимай "Next" на всех шагах (настройки по умолчанию подойдут)
5. После установки перезапусти PowerShell

## Способ 2: Через winget (если установлен)

```powershell
winget install --id Git.Git -e --source winget
```

## Проверка установки

После установки открой новый PowerShell и выполни:
```powershell
git --version
```

Должно показать версию Git, например: `git version 2.42.0`

---

## После установки Git

Вернись в папку проекта и выполни команды из инструкции!
